Task Orbit – Smart Task Manager
-> Project Description
Task Orbit is a beginner-friendly Task Management System built using Python.
It allows users to add, view, update, and delete tasks directly from the command line.
The project also stores tasks in a file, so data is saved even after closing the program.

This project is ideal for beginners who want to practice:

-> Python fundamentals

File handling

Lists and dictionaries

Functions

Conditional statements

Loop-based menu systems

-> Features
➕ Add new tasks

📋 View all tasks

✏️ Mark tasks as completed

❌ Delete tasks

⭐ Set task priority (High / Medium / Low)

💾 Data stored in file (Persistent storage)

🔁 Menu-driven system
